# Product Vision – Project Compass

## What is Project Compass?

Project Compass is a project management tool that helps users move from an idea to a structured project.

Instead of starting with an empty board, the user starts with a short project interview. The answers are used to create a project map with purpose, goals, deliverables, risks, decisions and next steps.

## Problem

Many project management tools start too late in the project process. They help users manage tasks, but they do not always help them understand the project.

Common problems:

- The project starts with tasks before the purpose is clear.
- Risks are not identified early.
- Decisions disappear in meetings, comments or chat messages.
- “Done” is unclear because quality criteria are missing.
- Teams lose the connection between daily work and the project goal.

## Product idea

Project Compass should help users answer:

- Why are we doing this?
- What should be achieved?
- What needs to be delivered?
- What can go wrong?
- What decisions are needed?
- What is the next important step?
- How do we know that the project succeeded?

## First target users

The first version is aimed at:

- students working in project groups
- small teams
- new project leaders
- associations and non-profit organizations
- small internal development projects

## Core promise

Project Compass helps users create structure before they start working.

It is not just a task board.

It is a project map.
